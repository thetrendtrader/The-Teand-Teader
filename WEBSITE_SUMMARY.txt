विकल्प 1: Direct (सबसे आसान)
  1. index.html को download करें
  2. अपने desktop/documents में save करें
  3. डबल-click करके खोलें
  4. Ready! 🎉

विकल्प 2: Local Server (Better)
  1. Terminal/CMD खोलें
  2. CD करें जहाँ index.html है
  3. Run करें:
     python -m http.server 8000
  4. Browser में जाएं:
     http://localhost:8000

विकल्प 3: Online Hosting
  1. index.html को किसी hosting पर upload करें
  2. URL access करें
  3. Done! 🌐

  1. जाएं: https://github.com/signup
  2. Sign up करें (email से)
  3. Email verify करें

  Already have account? सीधे login करें: https://github.com/login

  1. GitHub पर login करें
  2. Top-right में "+" देखेंगे
  3. Click करें → "New repository"
  4. Fill करें:
     
     Repository name: code-review-dashboard
     Description: Sudhir Yadav - Code Review Dashboard
     Public: ✅ (जरूरी है!)
     
  5. "Create repository" दबाएं

STEP 3️⃣: index.html Upload करें

  1. अपने repository पर जाएं
  2. "Add file" button दबाएं
  3. "Upload files" select करें
  4. index.html को drag करें या select करें
  5. "Commit changes" दबाएं

  ✅ Done! File upload हो गई

  1. Repository page पर जाएं
  2. "Settings" tab दबाएं (दाहिने)
  3. Left sidebar में "Pages" देखेंगे
  4. "Source" में set करें:
     
     Branch: main (या master)
     Folder: / (root)
     
  5. "Save" दबाएं

  ⏳ 30 seconds - 2 minutes रुकें...
